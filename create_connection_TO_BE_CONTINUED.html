Creating a database Connection with my pdo

There are two ways of accessing the database, PDO and my_sqli. You could use which ever you like. Both have their advantages.
  • PDO will work on 12 different database systems, whereas MySQLi will only work with MySQL databases.
  • So, if you have to switch your project to use another database, PDO makes the process easy.You only have to change the connection string and a few queries.With MySQLi on the other hand, you will need to rewrite the entire code - queries included.
  • Mysqli has both OOP and procedural. While my pdo only offers OOP.
  • Both support Prepared Statements. Prepared Statements protect from SQL injection, and are very important for web application security.
  • For me the best benefit of PDO is that it has an exception class to handle any problems that may occur in our database queries. If an exception is thrown within the try{ } block, the script stops executing and flows directly to the first catch(){ } block.
