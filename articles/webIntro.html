Getting started in web development

Posted by Jacob okoro on August 24, 2017

Web development involves developing a website for the website. The term development involves programming, documenting testing and bug-fixing involved in creating and maintaining a website (or framework).

It might involve developing static pages (Pages that involve only html and css) . To the most sophisticated web-based internet applications (web-app), e-business and social network services.

You should know though that threres a difference between a web developer and a web designer. A web developer is one who programs or codes applications for the world wide web.

Aspects of web development involves

  1. client-side scripting,
  2. server-side scripting
  3. data base technology

Client-side programming involves all programs run on the client (user’s computer or browser). It is used to make interactive pages. Make things happen dynamically on a page. Interact with the local storage. It carries out operations on the web browser whether mathematical or logical. Operations may be carries out on the client side because they require access to information or functionality that is available on the client but not on the server. Some times the user may need to obsrve the operations or provide input, also some operaitons need to be performed by the client eithout sending data to the server as it may take less time, use less bandwidth and incur less security risk.

To carry out client-side operations, one needs knowledge of:

  • Javascript
  • HTML
  • CSS
  • Jquery

Server-side

Server side programming really, involves work done on the back end of a website.Its usefulness lies in the need to host files and images in a database on a webserver (because the end-user’s computer is sometimes limited). It is a type of programming language that relies on code operated from the server of a wesite and not the computer that is actually accessing it. The bulk of the work in interpreting the code and running tasks Is done on the server and not the computer client (user’s browser). Here the code is embedded in order to allow for the computer to read it.

Server-side languages involvethe following
  • Asp.net
  • C
  • Java
  • PHP
  • python
  • Ruby
  • Node.js (a javascript framework)

It should be noted that client side development is done almost exclusively in javascript.

Database technologies

Finally as a web developer , you need to be familiar with a database technology. This is the technology involved in storing code, images, files and other data. A database is a computer based record keeping system which is used to record, maintain and retrieve data. A database management system is a collection of interrelated files and set of programs which allows users to store retrieve and modify information. It is responsible for creating , editing , deleting and generally maintaining a database or collection of daily records. There are several types of DBMS. But the most popular is relational dbms

Here are some examples of DBMS frameworks

  • Oracle
  • Microsoft SQL server
  • MySQL
  • IBM db2
  • Apache
  • Mongo db.

If you are like me that doesn't do much coding here, in it will be useful if you know the basics and know how to use the GUI of the application (if it has any). The one I use often is phpMyadmin.
